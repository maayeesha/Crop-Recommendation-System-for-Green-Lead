## 📋 Overview

The **Crop Recommendation System for Green Lead** is an intelligent, machine learning-powered solution designed to revolutionize agricultural decision-making. Built for [Green Lead Global](https://greenleadglobal.org/), this system leverages **Random Forest Algorithm** to provide farmers with personalized, data-driven crop recommendations based on their specific soil composition, weather conditions, and regional characteristics.

By analyzing key soil parameters (NPK ratios and pH levels) alongside climate data (temperature, humidity, rainfall), the system identifies the most suitable crops for optimal yield, sustainability, and environmental responsibility.

### 🌍 About Green Lead Global

This project is part of [Green Lead Global](https://greenleadglobal.org/)'s mission to advance sustainable agriculture and empower farmers with intelligent technology solutions. Learn more about their sustainability initiatives at [https://greenleadglobal.org/](https://greenleadglobal.org/)

## 🎯 Features

- 🌱 **Smart Crop Recommendations** - AI-powered suggestions based on NPK levels, pH, temperature, humidity, and rainfall
- 📊 **Data-Driven Insights** - Trained on comprehensive Indian crop recommendation dataset with real-world agricultural data
- 🗺️ **Region-Specific Analysis** - Customized recommendations for Central, Eastern, North Eastern, Northern, and Western India
- 📈 **Statistical Comparison** - Detailed statistics showing how your soil conditions compare to optimal ranges
- 🖼️ **Crop Information** - Visual crop data, descriptions, and farming insights
- 💻 **Intuitive Interface** - User-friendly Streamlit-based web application
- ♻️ **Sustainable Focus** - Promotes optimal crop selection aligned with [Green Lead Global](https://greenleadglobal.org/) sustainability principles
- ⚡ **Real-Time Predictions** - Instant recommendations powered by trained Random Forest model

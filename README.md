## 📋 Overview

The **Green Lead Project** is an intelligent crop recommendation system powered by **Random Forest Algorithm**. It analyzes soil composition, weather conditions, and regional location to provide data-driven recommendations for optimal crop selection.

Farmers can input key parameters about their soil and local climate, and the system will recommend the most suitable crop to maximize yield and sustainability.

## 🎯 Features

- 🌱 **Smart Crop Recommendations** - Based on soil NPK levels, pH, temperature, humidity, and rainfall
- 📊 **Data-Driven Predictions** - Trained on comprehensive Indian crop recommendation dataset
- 🗺️ **Region-Specific Insights** - Considers different regions: Central, Eastern, North Eastern, Northern, and Western India
- 📈 **Statistical Analysis** - Displays detailed statistics for recommended crops
- 🖼️ **Visual Crop Information** - Shows crop images and descriptions
- 💻 **User-Friendly Interface** - Built with Streamlit for easy interaction

## 🛠️ Technology Stack

- **Machine Learning**: scikit-learn (Random Forest)
- **Backend**: Python
- **Frontend**: Streamlit
- **Data Processing**: Pandas, NumPy
- **Model Serialization**: joblib
